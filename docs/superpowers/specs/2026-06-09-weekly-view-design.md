# 누디투두 v2 — 주간 뷰 디자인 스펙

작성일: 2026-06-09

---

## 개요

기존 `collapsed → expanded` 2단계 흐름에 `weekly` 상태를 추가한다.
데스크 위클리 플래너처럼 월~일 7컬럼이 가로로 펼쳐지는 주간 뷰.

---

## 상태 흐름

```
collapsed (180×180)
    ↓ 클릭
expanded (280×440)
    ↓ [이번 주] 버튼
weekly (1160×440)
    ↓ [접기] 버튼 또는 바깥 클릭
expanded (280×440)
    ↓ 바깥 클릭
collapsed (180×180)
```

- `collapsed ↔ weekly` 직접 전환 없음 — 항상 expanded 경유
- weekly → collapsed 직접 전환 없음
- weekly 바깥 클릭 → expanded로 복귀 (collapsed까지 가지 않음)

### [이번 주] 버튼 위치
expanded 뷰 카드 헤더 우측에 추가. 기존 `−` 접기 버튼 옆.

---

## 창 크기

| 상태 | 크기 |
|------|------|
| collapsed | 180 × 180 |
| expanded | 280 × 440 |
| weekly | 1160 × 440 |

**위치 앵커:** 화면 우측 하단 고정.
weekly로 펼쳐질 때 오른쪽 끝은 그대로, 왼쪽으로 넓어짐.

---

## 데이터 모델

요일 키 기반. 날짜 없이 요일별로 계속 축적.

```js
// electron-store에 저장되는 구조
{
  "mon": [{ id, text, done }, ...],
  "tue": [{ id, text, done }, ...],
  "wed": [...],
  "thu": [...],
  "fri": [...],
  "sat": [...],
  "sun": [...]
}
```

- 매주 같은 요일 칸에 투두가 계속 쌓임 (히스토리 축적)
- 주 이동 기능 없음 — 항상 현재 주 기준으로 오늘 요일 컬럼 하이라이트
- **마이그레이션:** 기존 저장된 투두(`todos` 키)는 앱 첫 실행 시 오늘 요일 키로 이관

## Weekly 뷰 레이아웃

```
[ weekly 상태 — 1160×440 ]
┌──────┬──────┬──────┬──────┬──────┬──────┬──────┐
│  월  │  화  │  수  │ [목] │  금  │  토  │  일  │  ← 상단 헤더 (요일+날짜)
├──────┼──────┼──────┼──────┼──────┼──────┼──────┤
│      │      │      │ ─□─  │      │      │      │
│      │      │      │ ─□─  │      │      │      │  ← 투두 목록 (스크롤)
│      │      │      │      │      │      │      │
├──────┼──────┼──────┼──────┼──────┼──────┼──────┤
│ +추가│ +추가│ +추가│ +추가│ +추가│ +추가│ +추가│  ← 각 칸 입력창
└──────┴──────┴──────┴──────┴──────┴──────┴──────┘
[ 캐릭터 + 말풍선 ]                        [ − 접기 ]
```

### 상세 규칙

- **컬럼 너비:** 각 160px, 7컬럼 = 1120px + 양쪽 패딩 = 1160px
- **오늘 컬럼:** 배경색 또는 테두리로 하이라이트
- **헤더:** 요일 + 날짜 표시 (예: `목 12`)
- **투두 목록:** 세로 스크롤 가능, 텍스트 말줄임(`...`) 처리
- **입력창:** 각 컬럼 하단 고정, Enter로 추가
- **캐릭터:** 하단 영역에 Rive 캐릭터 + 말풍선 (expanded와 동일 방식)
- **접기 버튼:** 우측 상단, 클릭 시 expanded로 복귀

---

## 드래그앤드롭

- **대상:** 투두 아이템
- **동작:** weekly 뷰 안에서 다른 요일 컬럼으로 드래그하면 해당 요일 키로 이동
- **구현:** HTML5 Drag and Drop API 사용
- **피드백:** 드래그 중 대상 컬럼 하이라이트, 드롭 시 즉시 이동

---

## Collapsed 상태 변경

- 기존: 전체 todos 기준 미완료 개수
- 변경: 오늘 요일 키(`today's day key`) 기준 미완료 개수
- 말풍선 텍스트: 동일 (`할 일 N개` / `할 일 없음`)
